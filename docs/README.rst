PMACE
-------------------------------------------------------------------
**ptycho_pmace** is an easy to use python-based library for PMACE framework for distributed reconstruction problems.


System Requirements
-------------------
1. Python>=3.8


Optional System Requirements
----------------------------
We also recommend:

* Installation using conda environment
